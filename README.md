# BrewMaster Franchise Landing Page

A premium coffee franchise landing page built with modern web technologies and responsive design.

## 📋 Overview

BrewMaster is a professional coffee franchise landing page designed to attract potential franchise partners. The website features a clean, modern design with responsive layouts that work seamlessly across all devices.

## 🚀 Features

- **Responsive Design**: Mobile-first approach with desktop optimization
- **Modern UI**: Clean typography, consistent spacing, and professional appearance
- **Working Navigation**: Functional hamburger menu for mobile devices
- **Hero Section**: Eye-catching layout with optimized image display
- **Business Overview**: Premium split layout with clear value proposition
- **Process Steps**: Clear progression showing franchise journey
- **Testimonials**: Social proof with customer success stories
- **Contact Form**: Professional form with validation and feedback

## 🛠 Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with Flexbox and Grid
- **Vanilla JavaScript**: No frameworks, pure JavaScript functionality
- **Responsive Design**: Mobile-first approach with desktop optimization

## 🎨 Design System

### Colors
- **Primary Brown**: `#6f4e37` (brand color)
- **Secondary Brown**: `#5a3e2b` (text and accents)
- **Light Beige**: `#faf8f6` (backgrounds)
- **White**: `#ffffff` (cards and text)

### Typography
- **Headings**: Clean, bold hierarchy
- **Body Text**: Optimized for readability
- **Responsive Scaling**: Proper font sizes for all devices

### Spacing
- **Consistent Scale**: Using rem units throughout
- **Mobile Optimization**: Tight spacing for small screens
- **Desktop Enhancement**: Balanced spacing for larger screens

## 📱 Responsive Breakpoints

- **Mobile**: `max-width: 768px`
- **Desktop**: `min-width: 769px`
- **Tablet**: Handled within mobile-first approach

## 🔧 Key Components

### Navigation
- **Desktop**: Horizontal layout with logo and navigation links
- **Mobile**: Hamburger menu with slide-in functionality
- **Sticky Header**: Fixed positioning for easy access

### Hero Section
- **Desktop**: Side-by-side layout (text left, image right)
- **Mobile**: Stacked layout (text top, image bottom)
- **Image Optimization**: Proper aspect ratio and loading

### Business Overview
- **Split Layout**: 2-column design (content left, image right)
- **Bullet Points**: Clean, minimal design with icon indicators
- **Professional Content**: Clear value proposition and benefits

### Process Steps
- **Numbered Progression**: Clear 1-2-3-4 step layout
- **Mobile Stacking**: Vertical layout on small screens
- **Desktop Grid**: Horizontal layout on larger screens
